Performance Review System
Современная система оценки эффективности сотрудников для HR-отделов и руководителей команд. Проект разработан на стеке React с использованием TypeScript/JavaScript, Ant Design и CSS Modules.

🚀 О проекте
Система позволяет проводить циклы оценки персонала: от постановки целей до финального подведения итогов руководителем. Реализована гибкая ролевая модель доступа (Employee, Manager, HR).

Основные возможности:
Управление целями: Постановка индивидуальных KPI сотрудником и их согласование.
Аналитика: Визуализация прогресса команды через графики (Recharts) и статистические карточки.
Обратная связь 360 градусов: Сбор мнений коллег, самооценка и оценка руководителя.
Подведение итогов: Фиксация результатов встречи и планирование развития.
Роли пользователей: Разграничение прав доступа к разделам системы.

Демонстрация проекта Посмотреть работающий пример можно здесь: (https://dmitryshpilkin.github.io/performance_review/)

📁 Структура проекта
Копировать
src/
├── api/              # Заглушки API запросов
├── components/       # Переиспользуемые компоненты (Navbar, GoalForm...)
├── context/          # Контекст авторизации пользователя
├── hooks/            # Кастомные хуки (логика аналитики)
├── pages/            # Экранные компоненты (Goals, Dashboard, Results...)

💡 Функционал по ролям
Роль	              Доступный функционал
Employee	          Просмотр своих целей, заполнение Self-Assessment.
Manager	              Оценка своих подчиненных, просмотр аналитики отдела, утверждение целей.
HR	                  Полный доступ ко всем данным компании, аналитика всей организации.

🚩 Известные особенности (Known Issues)
На данный момент используется "заглушечный" бэкенд (Hardcoded data). Данные сохраняются только в памяти браузера (или localStorage при доработке).
[Опционально] Бургер-меню требует дополнительной настройки кликов вне области для закрытия.

▶️ Запуск локально
Клонируйте репозиторий:
bash
git clone <https://github.com/DmitryShpilkin/performance_review>
Установите зависимости:
bash
npm install
Запустите сервер разработки:
bash
npm start
Откройте браузер по адресу http://localhost:3000
